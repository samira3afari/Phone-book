# 📞 دفترچه تلفن

یک برنامه وب برای مدیریت مخاطبان ساخته شده با Flask.

## امکانات
- اضافه کردن مخاطب جدید
- حذف مخاطب
- جستجو در مخاطبان (بر اساس نام یا شماره)
- ذخیره دائمی در فایل متنی

## تکنولوژی‌ها
- Python 3
- Flask
- HTML & CSS

## نحوه اجرا
1. Flask را نصب کن: `pip install flask`
2. فایل `app.py` را اجرا کن
3. در مرورگر برو به `http://127.0.0.1:5000`
